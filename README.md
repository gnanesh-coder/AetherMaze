# Aether Maze Visualizer

A high-performance, visually stunning algorithm visualizer for maze generation and pathfinding. Built with Electron and Vite, it demonstrates DAA (Design and Analysis of Algorithms) concepts through recursive backtracking and multi-path DFS/BFS solvers with a modern glassmorphic interface.

## Features

- **Algorithm Visualization**: 
  - **Recursive Backtracking**: For perfect maze generation.
  - **DFS (Depth-First Search)**: For finding multiple solution paths simultaneously.
  - **BFS (Breadth-First Search)**: For finding the guaranteed shortest path.
- **Modern Tech Stack**: Built using Electron, Vite, and Vanilla CSS.
- **Interactive UI**:
  - Adjustable animation speeds and grid sizes.
  - Drag-and-drop start and target points.
  - Multi-path rendering with distinct color palettes for better clarity.
- **Responsive Design**: Premium dark-mode aesthetic with smooth transitions and glassmorphism.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/aether-maze-visualizer.git
   cd aether-maze-visualizer
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm run electron:dev
   ```

## Development

- `npm run dev`: Start the Vite development server.
- `npm run build`: Build the project for production.
- `npm run electron:dev`: Run Electron in development mode.
